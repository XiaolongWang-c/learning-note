ResNet 为什么更容易训练

这个动画页专门解释一个核心问题：深层网络不是因为“最后分类难”，而是因为前面层的梯度在反向传播中变弱，导致浅层学不动。残差连接把“纯连乘”变成“恒等通路 + 小扰动”，所以信息和梯度都更容易通过。

普通深层网络：h = F(x)
残差网络：h = x + F(x)
核心：让梯度不至于消失
前向传播里的差别
普通层： h = F(x)
残差层： h = x + F(x)
反向传播里的关键差别
普通层： ∂L/∂x = ∂L/∂h · ∂F/∂x
残差层： ∂L/∂x = ∂L/∂h · ( I + ∂F/∂x )
这里的 I 是最关键的一项。它意味着：即使 ∂F/∂x 很小，梯度也还能沿着恒等路径传回去，而不是完全被层层连乘压扁。

为什么“最后全连接分类”并不是主要难点

如果深层特征真的学得好，最后的线性层或全连接层通常并不难。真正的问题在于：普通深层网络中，前面的特征层未必能被有效训练出来。一旦浅层参数几乎不更新，后面拿到的中间表示就不够好，最后分类自然也不会好。

为什么“skip connection”不只是参考

skip connection 不是“把上一层结果给下一层参考一下”，而是直接参与输出：h = x + F(x)。因此它不是辅助信息，而是输出的一部分；同时这条恒等路径也会在反向传播时为梯度提供直达通路。

为什么可以把残差看成“增量更新”

每个残差块不必重新定义一个全新的表示，它只需要回答：“在已有表示 x 的基础上，我还需要改多少？” 这就是残差 F(x)。如果某一层没有必要做复杂变换，学成 F(x) ≈ 0 就可以退化为恒等映射。

为什么普通网络在深层时更难优化

反向传播中梯度要经过链式法则连乘：∂L/∂x = ∂L/∂h · Π(∂hᶩ/∂hᶩ⁻¹)。若每层导数范数平均略小于 1，那么层数变深后整体就会指数级衰减。越靠前的层，拿到的梯度越弱，参数更新越无效。

普通深层网络：为什么梯度会弱

∂L/∂x = ∂L/∂h · ∂Fₙ/∂hₙ₋₁ · ∂Fₙ₋₁/∂hₙ₋₂ · ... · ∂F₁/∂x

这是一个“很多项连续相乘”的结构。如果这些导数项平均略小于 1，那么随着层数增加，最终传回最前面的梯度会迅速衰减。因此不是网络不能表达，而是训练过程中前层拿不到有效信号。

残差网络：为什么梯度更容易保住

h = x + F(x), ∂h/∂x = I + ∂F/∂x

关键是多了单位映射 I。它相当于告诉梯度：就算变换分支 F 当前学得一般，也还有一条直接路径能把信号传回去。所以残差连接并不是保证“梯度完全不变”，而是显著降低“梯度消失”的风险。

你可以这样记住

普通深层网络的问题不是“最后分类太难”，而是“前面层越来越学不动”。
残差连接也不是“拿上一层参考一下”，而是让 x 直接成为输出的一部分，于是每一层都更像：旧表示 + 小修正。
这就是为什么 ResNet 解决的不是“层数更多”本身，而是深层网络为什么难优化。
